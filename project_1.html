Projects> Cake Builder

TL;DR

This project demonstrates website development using agile methodology. Our group recommended a cake builder to automate the online ordering process for a local baker, allowing customers to design their orders and freeing up the client for other tasks. The project was completed in 3 weeks, and the website was built using HTML, CSS, JS.

Made With:

  • HTML
  • CSS
  • JS

Team Members:

  • Arik Blaushtein ○ UI Designer
  • Austin Chiatto ○ Web Developer
  • Inhye Yoon ○ UI/UX Designer
  • Michael Cheung ○ UX Designer

The development of the cake-building website was a challenging and rewarding experience. By using a combination of HTML, CSS, and JavaScript, we were able to create a visually appealing and interactive cake builder that allows users to easily design and order custom cakes online. The use of CSS and JavaScript to change images based on buttons changing classes was a unique feature that set this project apart from others in the class.

In future iterations of this project, I would like to update the Javascript functions to be less repetitive and capitalize on more efficient methods. Such as setting up Arrays to contain the different images and concatenation to replace the syntax when needed instead of rewriting entire lines of code to achieve the same thing.
